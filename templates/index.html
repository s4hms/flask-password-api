Password Generator API Documentation

A simple API to generate Password, consisting of Alphabets(lowercase, uppercase), numbers and special characters.

Note:

The special characters '#', '&' and '+' have to be passed in api URL as '%23', '%26' and '%2B' respectively.

Endpoint

Definition

uc:

Number of uppercase characters in the Password.

lc:

Number of lowercase characters in the Password.

nc:

Number of numeric characters in the Password.

sc:

Special characters in the Password.

Examples:

1) To generate Password containing 4 uppercase characters (uc), 8 lowercase characters (lc), 2 numeric characters (nc) and '^' and '#' (# has to be passed as %23) as special characters (sc): "http://localhost:5000/generate?uc=4&lc=8&nc=2&sc=^%23"

2) To generate Password containing 6 lowercase characters (lc), 4 numeric characters (nc) and '&' (%26) and '+' (%2B) as special characters (sc): "http://localhost:5000/generate?uc=4&lc=8&nc=2&sc=%26%2B"
